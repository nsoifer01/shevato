What we do

REST APIs, microservices, and the boring pieces that hold them together. Contract-first design, real testing, and code that the next engineer can actually read.

  • Spring Boot

Frontend and BFFs

Backend-for-frontend services that shape data for the web team, plus hands-on work in legacy JSP, vanilla JS, and modern static frontends when a project needs it.

Cloud and DevOps

Maven release pipelines, base-image cutovers, JFrog Artifactory, Jenkins, observability wiring. Whatever it takes to get a service from a feature branch to production safely.

  • Maven

  • Netlify

Databases and Legacy Integration

Oracle, PostgreSQL, mainframe interop, Firestore. We have spent a lot of time on the seam between a modern Spring Boot service and a system that has been around for decades.

  • Firestore
